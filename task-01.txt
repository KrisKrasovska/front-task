Написать что выводит данный код. Предложите 2 варианта модификации кода, чтобы ответ был следующим: Bad: 10, Bad: 12, Good: 15, Good: 21 
const arr = [10,12,15,21];
for (var i=0; i<arr.length; i++) {
	setTimeout(function() {
console.log(arr[i] > 13 ? `Goog:${arr[i]}`:`Bad:${arr[i]}` )
	}, 3000)
}
Данный код выведет 4 раза Bad:undefined, так как переменная i у нас задана через var и у нас есть setTimeout. Когда цикл из основного стека завершится и event loop перейдет к очереди с setTimeout, var будет равен 4, а в массиве элемента с индексом 4 нет - следовательно undefined.

1. Первый вариант модификации - это изменить var на let.
for (let i=0; i<arr.length; i++) {
	setTimeout(function() {
console.log(arr[i] > 13 ? `Goog:${arr[i]}`:`Bad:${arr[i]}` )
	}, 3000)
}
2. Второй вариант - использовать замыкание, чтобы захватить текущее значение переменной i в цикле.
for (var i = 0; i < arr.length; i++) {
  (function (i) {
    setTimeout(function () {
      console.log(arr[i] > 13 ? `Good:${arr[i]}` : `Bad:${arr[i]}`);
    }, 3000);
  })(i);
}
